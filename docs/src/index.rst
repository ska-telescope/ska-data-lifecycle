SKA Data Lifecycle Management
=============================

Welcome to the official documentation for the SKA Data Lifecycle Management (DLM) system;
developed and maintained by the Data Management (DM) team within the Data Processing ART.
This site provides an overview of DLM components, user-facing APIs, deployment options,
and practical guides for interacting with the DLM through both the request-based and event-driven interfaces (ska-dlm-client).
For high-level design details, see the `Solution Intent page <https://confluence.skatelescope.org/pages/viewpage.action?pageId=159387101>`_.
